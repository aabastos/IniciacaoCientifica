	Eficiência significa algo ser produtivo em uma determinada tarefa. Quando se deseja trabalhar com análises de dados, um fator importante é a eficiência na coleta de dados, sendo que, os dados é uma matéria prima vitar para que seja realizado o processo. Então, se torna importante que seja feita a determinação das melhores maneiras de serem realizadas tais aquisições de dados. Pesquisas de como as ferramentas funcionam e testes para medir o desempenho delas são importantes para que seja escolhida a técnica que será utilizada.
	O artigo irá mostrar as principais formas de coleta de dados utilizados em trabalhos anteriores relacionados à esse.
